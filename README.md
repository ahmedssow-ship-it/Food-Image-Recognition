# 🍲 Food Image Classification with Deep Learning

## 📌 Project Description

This project aims to build a **Deep Learning model** capable of recognizing different food dishes from images.

The objective is to classify each image into the correct food category using **Convolutional Neural Networks (CNNs)** and **Transfer Learning techniques**.

This project is part of a Computer Vision challenge focused on African and international food recognition.

---

## 📊 Dataset

The project uses food image datasets such as:

- :contentReference[oaicite:0]{index=0}  
- African food datasets (custom subset)

### Dataset Details:

- Multiple food categories (e.g. Thieboudienne, Yassa Poulet, Mafé, Jollof Rice)
- Images organized by class folders
- Train/Test split


---

## ⚙️ Data Preprocessing

- Resize images (224×224)
- Normalize pixel values
- Data augmentation:
  - Random Horizontal Flip
  - Random Rotation
  - Color Jitter
  - Random Resized Crop

---

## 🧠 Model Architecture

Two approaches were explored:

### 🔹 Baseline CNN
- Convolutional layers
- ReLU activation
- MaxPooling
- Fully connected layers
